# Pinterest 실행 가이드 (2026-07-04)

1~2시간 일회성 작업 → 방치 가능한 장기 자산. 판정: 1~2달 뒤 GA에 pinterest referral 잡히는지.

## 1. 계정 세팅 (15분)
- pinterest.co.kr → **비즈니스 계정**으로 가입 (또는 개인 가입 후 비즈니스 전환)
- 프로필 이름: `글씨방 — 무료 학습지 프린트` / 소개: "한글 따라쓰기·받아쓰기·수학 학습지를 무료로 만들어 인쇄하는 사이트. 회원가입 없음."
- 웹사이트: geulssibang.com 입력 → **"사이트 인증(클레임)"** 메뉴에서 HTML 태그 방식 선택
  → **나온 메타 태그를 Claude에게 전달** (전 페이지 head에 삽입해줌. 인증되면 핀에 로고 붙고 통계 제공)

## 2. 보드 3개 만들기
| 보드 이름 | 설명 |
|---|---|
| 한글 따라쓰기 학습지 | 자음 모음부터 이름·낱말까지, 무료로 출력하는 한글 쓰기 연습 프린트 |
| 초등 수학 학습지 | 덧셈 뺄셈 구구단 분수 문장제 — 매번 새 문제로 뽑는 무료 수학 프린트 |
| 받아쓰기·활동지 | 1·2학년 받아쓰기 시험지와 미로 찾기 활동지 무료 인쇄 |

## 3. 핀 15개 — 이미지 만들기 → 업로드
각 행: **생성 링크 클릭 → "이미지로 저장" 버튼 → 받은 PNG를 핀으로 업로드**.
연결 링크(대상)는 딥링크가 아니라 랜딩 페이지(설명·FAQ가 있어 방문자에게 나음).

업로드 요령: 하루 3~5핀씩 3일 분산 / 이미지는 받은 세로 PNG 그대로 / 제목·설명 복붙 OK.
