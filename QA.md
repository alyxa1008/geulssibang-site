# 글씨방 수동 QA 체크리스트 (2026-07-03)

코드 로직·링크·생성 규칙은 자동 검증 완료(./audit.sh). 여기는 **사람 눈·귀로만
확인 가능한 것**만 모음. 각 항목 앞 `[ ]`를 `[x]`로 바꾸며 진행.
⚠️ 시작 전: 각 페이지에서 **강력 새로고침**(PC: Cmd+Shift+R / 아이폰: 설정→Safari→방문기록 지우기 또는 사파리 새로고침 길게)

문제 발견 시: 페이지 주소 + 무엇이 어떻게 이상한지 + 스크린샷 → Claude에게 전달.

## A. 최근 변경분 (우선 — 폰에서 15분)

### A1. 수학 문장제 (/math/word/) ← 신규 도구, 최우선
- [ ] 아이 이름에 실제 이름 입력 → 문제에 그 이름이 자연스럽게 들어가는가 (조사 어색함 없나: "OO이는/OO는")
- [ ] 문장이 어색한 게 없는가 — 5~6번 "새 문제로 섞기" 눌러 훑기 (연필을 먹는다류, 이상한 숫자)
- [ ] 모바일에서 미리보기 시트가 화면에 맞게 축소되는가 (가로 스크롤 없어야)
- [ ] 문제지 인쇄·PDF: 문제가 페이지에 잘리지 않고, 정답지가 별도 장으로 나오는가
- [ ] 이미지로 저장: PNG에서 문장 줄바꿈이 자연스럽고 정답칸 줄이 안 겹치는가
- [ ] 링크 복사 → 새 탭에 붙여넣기 → 같은 문제·이름 그대로 복원되는가

### A2. 한글 글씨체 6종 (/hangul/)
- [ ] 글씨체 선택에 6종이 다 보이는가 (또박·손글씨·바탕·펜글씨·동글·감자꽃)
- [ ] 새 3종(펜글씨·동글·감자꽃)이 격자 칸을 벗어나지 않는가 — 받침 많은 글자로("뷁" 대신 "꿈, 빵, 학교")
- [ ] 새 글씨체로 이미지 저장 → PNG에도 그 글씨체가 적용돼 있는가 (폰트 로드 문제 확인)
- [ ] 인쇄 미리보기에서도 글씨체 유지되는가

### A3. 숫자 따라쓰기 (/hangul/suja/)
- [ ] 4개 버튼 각각 → 도구가 해당 내용(0~9 / 11~20 / 십 단위 / 하나둘셋)으로 열리는가
- [ ] "0~9" 버튼: 숫자가 한 글자씩 줄로 나오는가

### A4. 푸터 사이트맵 + 홈 (전 페이지 공통)
- [ ] 폰에서 홈 맨 아래 푸터: 4개 그룹(한글/받아쓰기/수학/놀이)이 읽기 좋게 줄바꿈되는가 (겹침·삐져나옴 없나)
- [ ] 깊은 페이지(예: /badaseugi/geupsu/1-1/)에서도 푸터가 똑같이 나오는가
- [ ] 홈 제목 아래 "✓ 회원가입 없음 · ✓ 인쇄 무제한 · ✓ 100% 무료"가 보기 좋게 표시되는가
- [ ] 홈 가이드 줄(한글/받아쓰기/수학 그룹)이 폰에서 어색하지 않은가

## C. 정기 전체 점검 (여유 있을 때 — 20분)

### C1. 받아쓰기 (/badaseugi/) — 간판 기능
- [ ] 폰 무음 해제 후 "시험 시작" → 번호+단어를 또박또박 불러주는가
- [ ] "다음 문제" 눌러 빨리 넘겨도 소리가 겹치지 않는가
- [ ] 시험 중 페이지를 나가면(뒤로가기) 소리가 멈추는가
- [ ] 시험지 인쇄: 문제지+정답지 장수·번호 정상

### C2. 인쇄 품질 (PC에서 — 실제 A4 또는 PDF로)
- [ ] 한글 따라쓰기: 격자가 페이지에 꽉 차고 잘리지 않는가
- [ ] 수학 세로셈: 문제 정렬·정답지 정상
- [ ] **어느 도구든 인쇄물에 광고·버튼·안내문이 찍히지 않는가** (애드센스 승인 후 재확인 필수)

### C3. PWA·공유
- [ ] 홈 화면 추가된 글씨방 아이콘 → 주소창 없이 앱처럼 열리는가
- [ ] 아무 도구에서 "링크 복사" → 카톡 나에게 보내기 → 미리보기 카드(제목·이미지)가 예쁘게 뜨는가

### C4. 콘텐츠 눈검사 (틈틈이)
- [ ] 급수표 단어에 오타·이상한 단어 없는가 (1-1, 1-2, 2-1, 2-2)
- [ ] 획순 페이지(/hangul/order/) 설명이 자연스러운가
