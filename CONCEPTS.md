# 개념 설명 시트 연구 노트 (2026-07-06 조사)

"개념 설명 함께 인쇄" 기능(ROADMAP 참조)의 제작 스펙. **교과서 도입 방식 그대로**가 원칙 —
어른식 설명(곱셈=빠른 덧셈)은 학교 방식과 충돌하므로 금지.

주 출처: 교육부 교원 연수 자료(KOCW/KERIS, 1학년 덧셈·뺄셈 지도 — 교과서 페이지 직접 인용),
2-1 곱셈 단원 구성(교과서 목차 기준). 지도 4단계 원칙(김수환 외): 생활 장면 → 모델로 나타내기
→ 조작으로 답 구하기 → 기호로 형식화.

## 시트 ① 덧셈 받아올림 (1-2 ~ 2-1)

교과서 순서 (이 순서 그대로 시트 구성):
1. **10의 짝꿍(보수)** — 가르기·모으기로 "몇과 몇이 만나 10"을 먼저 (1+9, 2+8 … 짝꿍표)
2. **10이 되는 더하기 / 10에서 빼기** — 수판(십칸판)에 채워 보이기
3. **10을 만들어 더하기** — 8+5는 → 8+2+3 → 10+3=13 (5를 2와 3으로 가르기).
   앞의 두 수가 10 되는 경우(8+2+4)와 뒤가 10 되는 경우(3+9+1) 모두
4. **세로셈 형식화** — 낱개는 낱개끼리 줄 맞춤(왜 줄을 맞추는지가 포인트), 받아올림 표기

시트 문구 예: "받아올림은 '10 만들기'예요. 8+5 → 8이 10이 되려면 2가 필요 →
5에서 2를 데려와 10 → 남은 3을 더해 13!"

## 시트 ② 뺄셈 받아내림 (1-2 ~ 2-1)

교과서 순서:
1. **제거 상황으로 도입** — "우유 27병에서 3병을 사 가면?" (실생활 장면 먼저)
2. **10에서 빼기** — 십의 보수 활용, 수판에서 지워 보이기
3. **(십몇)−(몇)** — 13−5: 10에서 5를 빼고(10−5=5) 남은 3을 더한다(5+3=8)
4. **세로셈 형식화** — 낱개 → 10개씩 묶음 순서로, 받아내림 표기

⚠️ 제작 전 확인 1건: 교과서가 두 방법(10에서 빼기 / 하나씩 이어 빼기)을 병행 제시하는지
최신 교과서 기준 재확인 후 반영. 시트는 "10에서 빼기"를 주 방법으로.

## 시트 ③ 곱셈 도입 (2-1 6단원 → 2-2 곱셈구구)

교과서 단원 순서 (확정):
1. **여러 가지 방법으로 세기** — 하나씩 → 묶어 세기(2씩, 5씩) → 뛰어 세기(3,6,9,12…)
2. **몇의 몇 배** — "2씩 3묶음 = 2의 3배"
3. **곱셈식** — 4씩 7묶음 = 4의 7배 = 4×7 = 28. 덧셈식(4+4+4+4+4+4+4)↔곱셈식 연결
4. (2-2) 곱셈구구 — 같은 단은 뛰어 세기와 같음 (7단 = 7씩 뛰어 세기)

시트 문구 예: "곱셈은 '묶음 세기'예요. 사탕이 4개씩 7묶음 → 4×7. ×는 '몇씩 몇 묶음'의 기호!"


## v1 제작 스펙 (7/13 배치)

- 시트 3종: 위 ①②③. A4 1장씩, 그림(수판·묶음 그림은 SVG/CSS로) + 큰 글씨 + 부모용 한 줄 팁
